## Descrição do Curso de Lógica de Programação

**DESCRIÇÃO E OBJETIVO DO CURSO:**

### Todos os exercicios desse curso são da plataforma: beecrowd.com

* Introdução à programação
* Expressões
    * Aritméticas
    * Comparativas
    * Lógicas
* Estrutura sequencial
    * Entrada de dados (comando de leitura)
    * Processamento de dados (comando de atribuição)
    * Saída de dados (comando de escrita)
* Estrutura condicional (`if`/`else`)
* Estruturas repetitivas (`while`, `for`)
* Vetores
* Matrizes

Este curso equivale à disciplina de Lógica de Programação, que é a disciplina inicial de programação em um curso de faculdade na área de Computação.
